
## Introduction
LLM visualizer is a small toolkit to visualize LLM.

## Quick start

### Installation

Run the following command to install the package.
```shell
git clone git@github.com:FateScript/llm_visualizer.git
cd llm_visualizer
pip install -v -e .  # or  python3 setup.py develop
```
